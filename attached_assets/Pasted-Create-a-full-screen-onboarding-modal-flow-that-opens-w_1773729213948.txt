Create a full-screen onboarding modal flow that opens when the user clicks “Start in 2 minutes” on the landing page.

This is not a static walkthrough.
It must feel like a real product demo with interaction, leading to signup.

🎯 GOAL

Show value before signup

Force user interaction (not passive slides)

Complete flow in under 60 seconds

Convert user to signup on final step

🧱 MODAL BEHAVIOR

Full-screen overlay (not small popup)

Centered content with max-width

Background slightly blurred

Smooth step transitions (slide + fade)

Progress bar at top (4 steps)

📊 PROGRESS BAR

Step 1 → 25%

Step 2 → 50%

Step 3 → 75%

Step 4 → 100%

🧠 UX RULES

No skip button

No navbar or distractions

Inputs should be pre-filled where possible

Each step must feel interactive, not informational

Total flow time < 60 seconds

🎬 ANIMATIONS

Step transition: slide + fade (200–300ms)

Button hover: slight scale

Input focus: soft glow

Success action: checkmark + highlight

📱 RESPONSIVE

Fully optimized for mobile

Full screen takeover

Sticky bottom CTA on mobile

⚙️ TECH NOTES

Maintain step state (1–4)

Persist demo actions temporarily

On signup success → redirect to dashboard

Preload demo data into dashboard for continuity

🚫 DO NOT

Do not make this a slideshow

Do not overload with text

Do not ask signup before demo

Do not add more than 4 steps

🎯 EXPECTED OUTPUT

A clean, modern, high-conversion onboarding modal UI that:

Feels like using the real product

Guides user step-by-step

Ends with a strong signup conversion
